About Application:

This application is built on Angular JS, 2DMap: Leafletjs 1.0 API and 3DMap: NASA World Window API

The images are loaded from tiles present at : https://d2xy2u667w9tvp.cloudfront.net/LAYDB-1-BlueMarbleWithBathymetry-data/1072915200000/

There are 5 levels of images in the pyramid.

Note:Work in progress on 3DMap page's World Window API image overlay. See Readme in the repo for more information.
